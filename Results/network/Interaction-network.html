Introduccin

Este trabajo tiene como objetivo explorar los cambios tempranos en los perfiles de expresin gnica asociados con el desarrollo del cncer colorrectal (CCR), que puedan ayudar en un futuro a identificar marcadores que optimicen los programas actuales de cribado. Para ello, se analizan los datos de los microarrays de miRNA de 3 muestras correspondientes a cncer colorrectal, 3 correspondientes a adenomas de alto grado, y 3 controles. En este apartado se realiza la integracin de los genes y miRNA diferencialmente expresados obtenidos en el anlisis de expresin diferencial.

A continuacin, se identifican los nodos con ms conexiones en la red miRNAs sobreexpresados-mRNA infraexpresados.

all_nodes <- unique(c(interaction1_clean$mature_mirna_id, interaction1_clean$target_symbol))

node_type <- ifelse(all_nodes %in% interaction1_clean$mature_mirna_id, "miRNA", "mRNA")
names(node_type) <- all_nodes

node_degree <- degree(g1, mode = "all")
miRNA_degree <- node_degree[names(node_degree) %in% interaction1_clean$mature_mirna_id]
mRNA_degree <- node_degree[names(node_degree) %in% interaction1_clean$target_symbol]

top_miRNA <- names(sort(miRNA_degree, decreasing = TRUE)[1])
top_miRNA_connections <- sort(miRNA_degree, decreasing = TRUE)[1]

top_mRNA <- names(sort(mRNA_degree, decreasing = TRUE)[1])
top_mRNA_connections <- sort(mRNA_degree, decreasing = TRUE)[1]

cat("El miRNA con ms conexiones es:", top_miRNA, "con", top_miRNA_connections, "conexiones.\n")
## El miRNA con ms conexiones es: hsa-miR-491-5p con 3 conexiones.
cat("El mRNA con ms conexiones es:", top_mRNA, "con", top_mRNA_connections, "conexiones.\n")
## El mRNA con ms conexiones es: PITPNC1 con 5 conexiones.
